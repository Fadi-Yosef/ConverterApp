# Converter App

Console-based Java application that converts values between different units.

## Features
This program includes three converters:

- Currency Converter (SEK → USD)
- Temperature Converter (Celsius → Fahrenheit)
- Length Converter (Meters → Kilometers)

## How It Works

When the program runs, the user sees a menu:

1. Currency Converter
2. Temperature Converter
3. Length Converter
4. Exit

After selecting a converter, the user chooses the conversion direction, enters a value, and the program prints the converted result.

## Technologies Used

- Java
- Maven
- IntelliJ IDEA
